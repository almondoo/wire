# セキュリティポリシー

## サポートバージョン

| バージョン | サポート状況       |
| ---------- | ------------------ |
| 1.0.x      | :white_check_mark: |
| < 1.0      | :x:                |

## 脆弱性の報告

本プロジェクトにセキュリティ上の脆弱性を発見した場合は、責任ある方法でご報告ください。

**セキュリティの脆弱性について、公開の GitHub Issue を作成しないでください。**

代わりに、[GitHub のプライベート脆弱性報告](https://github.com/almondoo/wire/security/advisories/new)からご報告ください。

報告後の対応目安：
- **48時間以内** に受領の確認
- **7日以内** にステータスの更新
- **30日以内** に確認された脆弱性に対する修正または緩和策の提示

## 対象範囲

本プロジェクトはコンパイル時のコードジェネレータです。ネットワーク通信、ユーザー認証、機密データの実行時処理は行いません。ただし、以下の点についてはセキュリティを重視しています：

- サプライチェーンセキュリティ（依存関係、ビルドプロセス）
- コード生成の正確性（生成されたコードが脆弱性を含まないこと）
- リポジトリおよび CI/CD のセキュリティ
